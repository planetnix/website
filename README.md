# Planet Nix Website

## Project Overview

A single-page website for Planet Nix, built with HTML and Tailwind CSS - no build steps required.

## Quickstart

1. Set up the environment with Flox
2. Start the live reload server
3. Edit `index.html` and see changes reload automatically

## Technology Stack

- **HTML5** - Single-page structure in `index.html`
- **Tailwind CSS** - Utility-first styling
- **Live Reload Tool** - Automatic browser refresh during development
- **Flox** - Development environment management

For detailed documentation, see [CLAUDE.md](CLAUDE.md).
